# AI Chat

To build and run the project:

```
git clone https://github.com/alejandro-du/vaadin-ai-chat.git
cd ChatBotSilab
mvn package
java -jar target/vaadin-chat-1.0-SNAPSHOT.jar
```
OR 
```
git clone https://github.com/alejandro-du/vaadin-ai-chat.git
cd vaadin-ai-chat
mvnw
```

And then go to http://localhost:8080/chat
